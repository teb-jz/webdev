
# Kalendarz

Stworzyć stronę internetową wyświetlającą kalendarz na bieżący miesiąc.

## Wymagania

* Wyświetlony jest rok, nazwa obecnego miesiąca oraz odpowiednio oznaczone kafelki reprezentujące jego poszczególne dni. Kafelek odpowiadający dzisiejszej dacie jest wyróżniony.

* Dostępne są dwa przyciski umożliwiające zmianę wybranego miesiąca.

* Po naciśnięciu na nazwę miesiąca wyświetla się lista umożliwiająca jego zmianę.

* Naciśnięcie roku umożliwia jego edycję.

* Po wybraniu danej daty pojawia się możliwość dodania notatki o wybranym kolorze i zawartości. Dane powinny zostać zapisane w przeglądarce przy pomocy `localStorage`.

* Po wybraniu notatki możliwe jest jej usunięcie oraz edycja.
